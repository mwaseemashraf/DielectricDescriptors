# Configuration-space sampling analysis

- Primary descriptor: shell-binned Ce/Zr cation pair-correlation vectors from the starting POSCAR structures.
- Dimensionality reduction: PCA of standardized pair-correlation vectors.
- Representative structures: deterministic k-means clusters in PC1-PC2 space; circled points are nearest to cluster centroids.
- Response overlays use relaxed/DFPT quantities only after sampling: ionic dielectric average, soft-mode descriptor, lattice anisotropy, and neighbor-distance variation.
- This supports a sampling claim, not a standalone dielectric-prediction claim.
